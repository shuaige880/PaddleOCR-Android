⚡ 快速命令参考：

Linux/WSL 用户：
  # 配置环境（首次）
  chmod +x setup_wsl.sh
  ./setup_wsl.sh
  
  # 构建 APK
  chmod +x build.sh
  ./build.sh
  
  # 或直接命令
  buildozer android debug
  
  # 安装到手机
  adb install bin/paddleocr-*.apk

Windows 用户：
  # 安装 WSL2
  wsl --install
  
  # 重启后运行
  .\setup_windows.bat
  
  # 在 WSL 中构建
  wsl
  cd ~/PaddleOCR-Android
  ./build.sh

测试 Kivy：
  python test_kivy.py

❓ 常见问题速查：

Q: Windows 能直接构建吗？
A: 不能，必须使用 WSL2 或 Linux 虚拟机。
   推荐使用 GitHub Actions 云构建，无需配置。

Q: 需要多长时间？
A: 首次构建 30-60 分钟，后续 5-10 分钟。

Q: 构建失败怎么办？
A: buildozer android clean
   rm -rf .buildozer
   buildozer android debug

Q: 如何安装到手机？
A: 方式1: USB 连接 + adb install
   方式2: 把 APK 发到手机，直接安装
   方式3: 微信/QQ 发送 APK

Q: 首次运行需要注意什么？
A: 需要联网下载 OCR 模型（20MB）
   请连接 WiFi 并等待 1-2 分钟
   下载后可完全离线使用
